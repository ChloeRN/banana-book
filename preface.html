Preface

0.1 Why this book?

Capture-recapture. HMM. And Bayes.

0.2 Who should read this book?

This book is aimed at two main audiences:

  • Beginners, but comfortable using R to do data analysis, and loops.

  • But also advanced analysts, frequentist framework, for Bayesian. Also others for HMM and open the full flexibility of framework. Uncertainty.

0.3 What will you learn?

The book is divided into two parts. Give description.

0.4 What won’t you learn?

No math ou presque. Don’t cover Bayesian statistics, or HMM fully. Suggested reading. There are also a number of important topics specific to capture-recapture that I don’t cover: closed-population, and spatial capture-recapture. Refer to books.

0.5 Prerequisites

This book uses primarily the R package NIMBLE, so you need to at least install R and the NIMBLE package. Also a bunch of R packages. You can install them all at once by running:

install.packages(c(
  "magick", "MCMCvis", "nimble", "pdftools", 
  "tidyverse", "wesanderson" 
))

0.6 Acknowledgements

This book was written in the open and chapters were advertised on twitter when complete. Read drafts, fixed typos, and suggested improvements. I’m deeply grateful for their help.

0.7 How this book was written

This book was written in RStudio using bookdown. The website is hosted with GitHub Pages, and automatically updated after every push by Github Actions. The complete source is available from GitHub.
